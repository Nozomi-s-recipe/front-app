## **成功の鍵となる判断ポイント**

### 各フェーズ終了時の判断基準

* **フェーズ1終了時** : 直帰率が改善しているか？検索機能の利用率は？
* **フェーズ2終了時** : 会員登録率20%達成？リピート率向上？
* **フェーズ3終了時** : DAUが安定して増加？
* **フェーズ4終了時** : エンゲージメント指標が目標達成？
* **フェーズ5以降** : コミュニティが自走しているか？

### ピボット検討ポイント

* フェーズ2で会員登録率が10%未満 → 機能の魅力不足
* フェーズ3でDAU増加が鈍化 → 利便性機能の優先順位見直し
* フェーズ4で献立機能の利用率が20%未満 → UI/UX再設計

このロードマップは柔軟に調整し、ユーザーフィードバックとデータを基に最適化していくことが重要です。
